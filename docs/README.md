# VEX API Documentation

This folder contains auto-generated API documentation from `nim doc`.

## Regenerating Docs

```bash
nim doc --project --outdir:docs src/vex/vex.nim
```

## Viewing

Open `vex.html` in a web browser, or use a static site generator.

## Structure

- `vex.html` - Main documentation for all VEX modules
- Individual module docs are also generated
